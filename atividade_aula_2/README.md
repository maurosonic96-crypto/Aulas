# Evolução dos Sistemas Operacionais

Resumo técnico expandido (500 linhas) baseado nas páginas iniciais de *Sistemas Operacionais Modernos(Tanembaum,Andrew S. e Bos,Herbert)*.

Este projeto organiza historicamente e conceitualmente a evolução dos sistemas operacionais, destacando o problema fundamental, o papel da abstração e o gerenciamento de recursos.

---

## 📌 Escopo

- Evolução histórica das gerações
- Problema fundamental dos SO
- Máquina estendida vs. gerenciador de recursos
- Transição: caos → automação → compartilhamento → complexidade

---

## 🧠 Problema Central

Como organizar recursos físicos limitados  
(CPU, memória, dispositivos)  
para múltiplos programas e usuários  
de forma eficiente e segura?

---
